## Prerequisites

- C compiler (GCC recommended)
- Basic understanding of C programming
- Text editor or IDE

## How to Use

1. Navigate to the project directory you want to work with
2. Compile the source code:
   ```bash
   gcc main.c -o program
   ```
3. Run the executable:
   ```bash
   ./program
   ```
